Git笔记(二)

1.查看提交日志.

git log命令从最近到最远的提交日志。git log –pretty=oneline简化显示


2.版本退回

HEAD 表示当前版本， HEAD^ 表示上一个版本，HEAD^^表示上上一个版本……HEAD~100表示往上100个版本。
